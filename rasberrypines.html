Los pines de alimentación se utilizan para alimentar el pico desde la fuente de alimentación o para alimentar los sensores y periféricos desde el pico.

El Pi Pico se puede alimentar a través de un cable USB (a través de VBUS) o mediante una batería u otras fuentes (a través de VSYS), cuando ambas entradas están conectadas juntas, la entrada con mayor potencial suministra la energía. Para conectar 2 fuentes, debe conectarlas a VSYS en serie con un diodo Schottky.

Uso de los pines


  • VBUS: 40
    • Es la entrada de 5 V del puerto micro-USB; si no hay un micro-USB conectado al pico, no habrá energía para el VBUS.

    VSYS: 39 Es el bus principal del sistema el que suministra la salida de 3,3 V al microprocesador RP2040 y otros pines IO. VSYS alimenta energía al RT6150 SMPS que genera una salida fija de 3.3V.

    3V3_ES: 37
    es el pin habilitado integrado, habilita la fuente de alimentación de 3,3 V en la placa pico, se puede utilizar externamente para encender o apagar el pico.

    3V3(Fuera): 36
    Es el pin de suministro del pin principal del RP2040 y su IO es generado por el SMPS, por lo que este pin se puede utilizar para habilitar el circuito de salida. La corriente máxima permitida es 300 mA.

    GND: 3,8,13,18,23,28,33,38
    Todos los pines de tierra de las placas están interconectados. Todos los dispositivos, periféricos y sensores están conectados a los pines de tierra para cerrar el circuito eléctrico y lograr una tierra de referencia común en todo el circuito. El tablero tiene un total de 9 pines de tierra. 8 en GPIO y 1 para depuración

    AGND: 33
    También conocido como tierra analógica. Está destinado al voltaje de referencia para los ADC y DAC. Es para la referencia de tierra para GPIO26-29, un plano de tierra analógico separado corre debajo de estas señales y termina en este pin. En ausencia de un ADC o ADC, este pin actúa como tierra digital.

    TP1: Ground
    TP2: USB DM
    TP3: USB DP
    TP4: Pin GPIO23/SMPS PS (no para usar)
    TP5: GPIO25/LED (no para usar)
    TP6: BOOTSEL

    Estos pines se pueden utilizar para fines tales como si la placa se utiliza como módulos de montaje en superficie. Se pueden utilizar TP1, TP2 y TP3 en lugar de micro-USB. TP6 se ​​puede utilizar para llevar el sistema al modo de programación USB de almacenamiento masivo (cortándolo al nivel bajo durante el encendido). TP4 y TP5 no están destinados a ser utilizados en absoluto.

    Tiene un pin adicional llamado Ejecutar.

    RUN : es un pin habilitado para RP2040. También se puede utilizar como reinicio del microcontrolador RP2040.
