¿Cómo se programa el Raspberry pi pico?


El método más sencillo para programar su Pi Pico es mediante MicroPython ; puede utilizar un compilador como Thonny IDE . Raspberry Pi Pico viene con dos plataformas diferentes para programación y también admite dos lenguajes de programación diferentes, es decir, " C/C++ o MicroPython".


Aqui el sitio para descargar


Como Raspberry Pi OS siempre tiene algunas versiones de Python atrasadas, la única manera de instalar la última versión de Python en su Raspberry Pi es descargar el código fuente desde el sitio web oficial. En la página de descargas, se enumeran las últimas versiones con su fecha de lanzamiento y estado de mantenimiento.
instalarlo manualmente:

  • Paso 1: Descarga la última versión de Python desde la web oficial.

  • Paso 2: Extrae los archivos en la Raspberry Pi.

  • Paso 3: Configurar el sistema para utilizar la última versión de Python.
